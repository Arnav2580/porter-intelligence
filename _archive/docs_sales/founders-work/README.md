- Use this pack together with [_archive/archive/checklist.md](../_archive/archive/checklist.md).

What this pack solves:
- It removes "blank page" risk for the founder.
- It makes the sale process transferable across sessions.
- It ensures the founder story stays aligned with the actual product state.
- It is grounded in real research on Porter's leadership, financials, and operations.

What this pack does not replace:
- Real product verification
- Real rehearsal (out loud, timed, with silence practice)
- Real legal review (have a lawyer review the NDA and MSA before printing)
- Real live-demo practice on the actual environment

Working rule:
- If the product state changes materially, update the matching day file before outreach.
